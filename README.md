# NFT-PORTAL
A place for users to mint NFT with their NRIC and wallet address

## Prerequisites
- Node.js ^v16.20.0
- Metamask
- Ganache UI or ganache-cli
- Truffle@latest framework
- InterPlanetary File System / IPFS
- React.js
- Solidity
- OpenZeppelin
